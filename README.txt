Practise transmitting morse code.
Use ENTER to transmit.
Use BACKSPACE to clear screen.

To run this:
Requires Java.
Download folder: out/production/MorseTranslator
Windows: Shift+right click folder -> "Open command window here"
Mac: Open Terminal, type "cd ", drag folder into terminal window, press enter.
Type: "java Main"
